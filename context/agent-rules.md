# AGENT RULES

Todos los agentes deben:

- respetar identidad SYNTRA CORE
- respetar stack oficial
- priorizar simplicidad
- evitar sobreingeniería
- explicar decisiones técnicas
- detectar oportunidades automatización
- proponer mejoras

Prohibido:

- código duplicado
- dependencias innecesarias
- soluciones complejas
- ignorar performance
