# SYNTRA CORE — Copywriting Rules

## Objetivo

Todo el copywriting de SYNTRA CORE debe transmitir:

* precisión
* sofisticación
* claridad
* tecnología
* innovación
* confianza
* minimalismo
* inteligencia operativa

La comunicación debe sentirse moderna, premium y técnica.

Nunca genérica.

---

# Tono de Comunicación

El tono oficial de SYNTRA CORE debe ser:

* profesional
* moderno
* minimalista
* tecnológico
* elegante
* seguro
* directo
* sofisticado

La comunicación debe evitar exageraciones innecesarias.

Debe transmitir autoridad mediante claridad técnica y simplicidad.

---

# Estilo de Escritura

## Priorizar

* frases claras
* impacto visual
* estructuras simples
* mensajes cortos
* claridad conceptual
* precisión técnica

---

## Evitar

* exceso de texto
* marketing agresivo
* frases vacías
* buzzwords innecesarios
* exageraciones
* tono infantil
* lenguaje informal excesivo

---

# Filosofía de Comunicación

SYNTRA CORE no vende “páginas web”.

SYNTRA CORE construye:

* sistemas digitales
* automatizaciones
* infraestructura inteligente
* experiencias premium
* ecosistemas escalables

La comunicación debe reflejar esa diferencia.

---

# Inspiración de Marca

Inspiraciones oficiales:

* Linear
* Vercel
* Stripe
* Framer
* Raycast
* Supabase

La comunicación debe sentirse:

* moderna
* limpia
* precisa
* tecnológica
* premium

---

# Reglas de Headlines

Los headlines deben:

* ser cortos
* tener impacto
* transmitir sofisticación
* evitar clichés

---

## Buenos ejemplos

* Build intelligent systems
* AI-native digital infrastructure
* Systems designed for scale
* Modern software experiences
* Intelligent automation for modern companies

---

## Evitar

* “Llevamos tu negocio al siguiente nivel”
* “La mejor agencia digital”
* “Transformamos ideas en realidad”
* “Soluciones innovadoras para empresas”

---

# CTAs

Los CTAs deben sentirse modernos y premium.

---

## Buenos ejemplos

* Start your system
* Build with SYNTRA
* Launch smarter
* Explore the ecosystem
* Design your infrastructure

---

## Evitar

* Contáctanos ahora
* Pedir presupuesto
* Solicitar demo urgente
* Comprar ahora

---

# Lenguaje Visual

El copy debe acompañar una estética:

* dark mode
* minimalista
* tecnológica
* elegante
* futurista

Nunca debe sentirse:

* corporativo tradicional
* genérico
* recargado
* agresivo

---

# UX Writing

La microcopy debe ser:

* simple
* clara
* elegante
* breve

---

## Ejemplos

Correcto:

* Loading system...
* Initializing workflow
* Processing automation
* System ready

Incorrecto:

* Estamos cargando la información para usted
* Su solicitud está siendo procesada actualmente

---

# Filosofía Final

SYNTRA CORE comunica como una empresa tecnológica moderna.

No como una agencia tradicional.

Cada palabra debe transmitir:

* inteligencia
* precisión
* sofisticación
* tecnología
* escalabilidad
* diseño premium
