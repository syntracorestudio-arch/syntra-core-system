# DEVELOPMENT RULES

Todo desarrollo debe cumplir:

- TypeScript estricto
- Mobile-first
- Componentes reutilizables
- Arquitectura modular
- Código limpio
- Accesibilidad WCAG AA
- Lighthouse +95
- SEO técnico obligatorio
- Manejo errores obligatorio
- Variables entorno obligatorias
- Evitar duplicación
- Documentación obligatoria

Prioridades:

1. Performance
2. Escalabilidad
3. Mantenibilidad
4. UX
