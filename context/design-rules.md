# SYNTRA CORE — Design Rules

## Objetivo

Toda interfaz construida bajo SYNTRA CORE debe transmitir:

- sofisticación tecnológica,
- minimalismo premium,
- claridad visual,
- sensación futurista,
- y alto rendimiento visual.

La estética debe sentirse moderna, limpia y profesional.

Nunca sobrecargada.

---

# Filosofía Visual

SYNTRA CORE utiliza una combinación de:

- minimalismo tecnológico,
- dark mode premium,
- glassmorphism moderado,
- motion elegante,
- y alto contraste visual.

La experiencia debe sentirse:

- rápida,
- fluida,
- inteligente,
- moderna,
- y escalable.

---

# Principios de Diseño

## 1. Performance Visual

Las interfaces deben sentirse ligeras y rápidas.

Evitar:

- exceso de efectos,
- animaciones innecesarias,
- sombras exageradas,
- blur excesivo,
- componentes pesados.

---

## 2. Minimalismo Inteligente

Menos elementos.
Más claridad.

Cada componente debe tener una función clara.

Eliminar ruido visual.

---

## 3. UX Premium

Las interfaces deben sentirse:

- refinadas,
- espaciosas,
- modernas,
- y profesionales.

Nunca “template genérico”.

---

## 4. Consistencia Absoluta

Todos los componentes deben mantener:

- mismo sistema de spacing,
- mismas sombras,
- mismas animaciones,
- mismas tipografías,
- misma lógica visual.

# Reglas de Layout

## Espaciado

Usar spacing amplio.

Evitar interfaces comprimidas.

Las secciones deben respirar visualmente.

---

## Grid System

Usar layouts modulares y consistentes.

Preferencia:

- grids de 12 columnas,
- max-width controlado,
- spacing uniforme.

---

## Mobile First

TODO debe diseñarse primero para mobile.

La experiencia móvil es prioridad absoluta.

---

# Reglas de Componentes

## Botones

Los botones deben:

- tener buen contraste,
- hover elegante,
- transición suave,
- tamaño cómodo,
- y apariencia premium.

Evitar:

- botones genéricos,
- colores saturados,
- estilos antiguos.

---

## Cards

Las cards deben:

- usar glassmorphism moderado,
- bordes suaves,
- sombras sutiles,
- separación clara.

---

## Inputs

Los formularios deben sentirse:

- modernos,
- minimalistas,
- rápidos,
- y accesibles.

---

# Motion Design

## Filosofía

Las animaciones deben:

- acompañar,
- no distraer.

Todo movimiento debe sentirse:

- suave,
- elegante,
- profesional.

---

## Reglas

Usar:

- fade-in,
- smooth transitions,
- soft hover effects,
- subtle motion.

Evitar:

- rebotes exagerados,
- efectos caricaturescos,
- animaciones agresivas.

---

# Glassmorphism

Permitido de forma MODERADA.

Usar:

- blur suave,
- transparencias ligeras,
- bordes delicados.

Nunca exagerar.

---

# Sombras

Sombras suaves y modernas.

Evitar sombras pesadas o antiguas.

---

# Bordes

Usar:

- rounded-xl
- rounded-2xl

Preferencia por esquinas suaves y modernas.

---

# SEO Visual

La interfaz debe priorizar:

- claridad,
- jerarquía,
- escaneabilidad,
- accesibilidad.

---

# Accesibilidad

Obligatorio:

- contraste adecuado,
- tamaños legibles,
- focus states visibles,
- navegación clara.

Objetivo mínimo:

WCAG AA.

---

# Reglas Técnicas

Frontend obligatorio:

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
- shadcn/ui

---

# Regla Final

Cada interfaz creada por SYNTRA CORE debe sentirse:

“como un producto tecnológico premium construido por una software factory moderna impulsada por IA.”
