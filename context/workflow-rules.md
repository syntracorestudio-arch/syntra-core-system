# WORKFLOW RULES

Todo workflow debe:

- incluir logs
- incluir manejo errores
- incluir documentación
- evitar pasos innecesarios
- priorizar reutilización

Todo workflow IA debe:

- validar entradas
- validar salidas
- controlar costos
- detectar fallos

Todo workflow n8n debe:

- ser modular
- usar webhooks correctamente
- registrar errores
