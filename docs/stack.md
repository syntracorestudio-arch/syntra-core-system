# SYNTRA CORE — Stack Oficial

## Filosofía Técnica

El stack de SYNTRA CORE fue diseñado bajo 5 principios:

1. Performance
2. Escalabilidad
3. Velocidad de desarrollo
4. Modularidad
5. Experiencia premium

Toda tecnología utilizada debe:

- ser moderna,
- escalable,
- mantenible,
- y compatible con workflows AI-Native.

---

# Frontend Oficial

## Framework Principal

Next.js

---

## Reglas

- App Router obligatorio
- Server Components por defecto
- Client Components solo cuando sean necesarios
- SEO-first architecture
- Mobile-first
- Arquitectura modular

---

## Librerías Oficiales

### React

Base UI.

---

### TypeScript

Obligatorio en modo estricto.

Nunca usar:

- any innecesario
- tipados débiles
- lógica insegura

---

### TailwindCSS

Sistema oficial de estilos.

---

### Framer Motion

Sistema oficial de motion design.

Uso:

- animaciones suaves,
- transitions,
- reveal effects,
- hover states.

---

### shadcn/ui

Sistema base de componentes.

Todos los componentes deben:

- personalizarse,
- mantener consistencia visual,
- y alinearse con SYNTRA UI.

## Filosofía IA

La IA debe:

- acelerar ejecución,
- mejorar calidad,
- reducir tareas repetitivas,
- y potenciar escalabilidad.

La IA NO debe:

- generar código desorganizado,
- romper arquitectura,
- duplicar lógica,
- ni ignorar estándares.
