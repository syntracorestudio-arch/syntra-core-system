# Principios de SYNTRA CORE

1. Performance First

Todo debe sentirse rápido y fluido.

2. Arquitectura Modular

Nada debe construirse dos veces.

3. UX Premium

Las experiencias deben sentirse modernas y profesionales.

4. Automatización por Defecto

Todo proceso repetitivo debe automatizarse.

5. Escalabilidad

Pensar siempre en crecimiento futuro.

6. Excelencia Técnica

Código limpio y mantenible es obligatorio.
