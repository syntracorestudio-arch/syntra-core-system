# SYNTRA CORE — Asset Rules

## Filosofía Visual

Todos los assets deben transmitir:

- tecnología,
- sofisticación,
- minimalismo,
- modernidad,
- y estética premium.

---

# Estilo Permitido

- dark mode
- glow effects
- gradients modernos
- glassmorphism moderado
- interfaces limpias
- elementos futuristas

---

# Estilo Prohibido

- imágenes genéricas corporativas
- stock photos cliché
- colores saturados excesivos
- diseño antiguo
- elementos caricaturescos

---

# Referencias Visuales

Inspiración oficial:

- Linear
- Vercel
- Stripe
- Raycast
- Framer
- Supabase

---

# Reglas de Imágenes

Toda imagen debe sentirse:

- premium,
- moderna,
- tecnológica,
- minimalista.

---

# Reglas de Mockups

Los mockups deben:

- mostrar interfaces modernas,
- usar dark mode,
- tener glow suave,
- mantener consistencia visual.

---

# Reglas de Branding

Todos los assets deben respetar:

- paleta oficial,
- tipografía oficial,
- y dirección visual de SYNTRA CORE.
